What you're seeing: The left panel shows a regular grid in input space, colored by position. The middle panel shows how the network deforms that grid — each layer applies a linear transform (rotation + stretch) then an activation function (fold/crease). Use the morph slider to smoothly interpolate between layers. The right panel shows the decision boundary the network has learned.
How activations warp space: Without activation functions, every layer is just a linear map (rotation + scaling). Activations introduce nonlinearity — they fold and crease the space. ReLU sets everything below 0 to 0 (a hard fold). Sigmoid squashes into (0,1). Tanh squashes into (-1,1). GELU applies a smooth probabilistic gate. Use the slider to smoothly transition from the linear pre-activation to the full nonlinear activation.
Loss surface as a function of two weights — gradient descent paths shown as colored trails
Loss landscapes are geometric: Training a neural network = finding the lowest point on a high-dimensional surface. This view shows loss as a function of just 2 weights. Different random initializations (colored balls) follow different gradient paths and can end up in different local minima. Sigmoid activations create smooth rolling hills; ReLU creates sharp ridges and creases.
Words positioned by meaning — closer = more similar. Lines connect related pairs.
Meaning IS geometry: In a trained language model, words with similar meanings cluster together in high-dimensional space. Directions encode relationships: king - man + woman ≈ queen. This 2D projection (via PCA or t-SNE) preserves the essential structure. The distances between points reflect semantic similarity — the geometry of language.
