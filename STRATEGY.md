# MathTools Strategy: Free Viral Tools → Paid Heavy Use

## The Model
1. Build small interesting tool
2. Host free (GitHub Pages)
3. Share on X with thread (the story matters more than the tool)
4. Free tier drives virality
5. Paid tier (Stripe) captures the 1-3% who use it for real work
6. Repeat with new tool every 2-3 days
